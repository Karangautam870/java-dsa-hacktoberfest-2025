2. Create branch: git checkout -b add-<algorithm>-<yourname>
3. Add Java file in correct folder with header:
   - Description
   - Time Complexity
   - Space Complexity
   - Example main()
4. Commit, push, open PR: Title "Add <Algorithm> in Java"
